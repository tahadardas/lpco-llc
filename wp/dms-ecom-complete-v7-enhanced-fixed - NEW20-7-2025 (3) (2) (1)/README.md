## 🚀 البدء السريع

### 1. التثبيت
```bash
# احتفظ بنسخة احتياطية
cp includes/import-export-products.php includes/import-export-products.php.backup

# استبدل الملف الجديد
cp new-import-export-products.php includes/import-export-products.php
```

### 2. الوصول
```
WordPress Admin → DMS ECommerce → استيراد / تصدير المنتجات
```

### 3. الاستخدام
```
📤 تصدير → اختر النوع → احفظ الملف
📥 استيراد → اختر الملف → انقر الاستيراد
```
